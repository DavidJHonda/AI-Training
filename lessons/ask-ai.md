## BUILD YOUR SKILLS

# Ask AI

You’ve set the tool up. The next skill is what to do when you’re not sure how to use it for something. The answer is almost always the same: ask the AI.

It has read millions of examples of good prompts, task setups, and questions, across nearly every subject. So when you’re stuck on how to ask, what it needs, or where to start, the AI itself is usually a better source than your own first guess.

## Four ways to ask AI for help using AI

When you’re stuck on the how, not the what, these four moves get you unstuck fast:

- “Write me a strong prompt for ___, then I’ll edit it.”
- “What do you need from me to do ___ well?”
- “How would you approach ___? Give me a few options.”
- “What should I be asking you about this that I haven’t?”
The pattern is the same each time: instead of guessing how to use the tool, you make the tool show you. It was trained on how-to-use-AI writing and on the subjects themselves, so the how-to layer is exactly where it’s strong.

🔑 **It’s still AI, so keep the call yours.**Asking AI how to ask gets you unstuck faster, but its advice can be wrong like any other answer. Use it to start, then check what comes back and decide for yourself.
