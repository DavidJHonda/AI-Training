🗣️ NARRATIVE ARC:
Hook: AI has superpowers no other software has, for a simple reason: it's built on a different foundation. Normal software is written line by line — rules, most commonly IF-THEN-ELSE: if the password matches, open the app; else, show an error. Your calculator and your GPA portal run the same rules the same way every time. AI is built on patterns instead. Use the lesson's cooking analogy: normal software is a robot cooking from a cookbook — someone wrote every step, same dish every time. AI is a chef: no cookbook, thousands of dishes cooked, patterns absorbed, so it can handle a dish it's never made. Same kitchen, completely different way to get to dinner. Show the difference on one question — "What's the best game for my new PS5?" Normal software returns the same preset list no matter who asks. AI builds a fresh answer shaped to your exact question. Second difference: structure. Normal software needs clean rows, fields, and labels. AI takes messy human input — scribbled notes, rambling voice memos, photos, half-formed questions — and turns it into something usable. But the foundation cuts both ways: ask normal software twice, same answer, and when it's wrong there's a bug on one line you can find and fix. Ask AI twice, the answer will likely change, and when it's wrong it's confidently wrong with no line to point to. Close on the trade: rock-solid consistency on one side; handling the unfamiliar and making sense of mess on the other. Knowing which foundation you're standing on tells you which jobs to hand each one.
